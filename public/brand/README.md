# EXVIASS S.A. Brand Assets

- `exviass-logo.png`: logo principal horizontal.
- `exviass-app-icon.png`: propuesta de ícono moderno para app móvil, ruta fija y transporte compartido.
- `exviass-van-hero.png`: imagen de van para hero y tarjetas de turnos.

Use this asset through `/brand/exviass-logo.png` in Next.js components.

Recommended use:
- Headers and auth screens: full horizontal logo.
- Small icons/favicon: use `exviass-app-icon.png`.
- Passenger flow hero/trip visuals: use `exviass-van-hero.png`.

Car assets for trip cards:
- `/cars/avanzanegro.webp`, `/cars/avanzarojo.webp`, `/cars/avanzaverde.webp`: originals.
- `/cars/transparent/*-transparent.png`: background-removed versions for card overlays.
